##模块 Module

模块是一个包含有一系列数据、函数、类等组成的程序组

模块是一个文件，模块文件名通常以.py结尾

**作用**

让一些相关的数据、函数、类等有逻辑的组织在一起，使逻辑结构更加清晰

模块中的数据、函数和类等可提供给其他模块或函数使用

**模块的分类**

+ 内置模块（builtins），在解释器的内部可以直接使用

+ 标准库模块，安装python时已安装且可直接使用

+ 第三方模块（通常为开源）,需要自己安装

+ 用户自己完成的模块（可以作为其它人的第三方模块）


##模块的导入语句

**import语句**

```python
import 模块名1[as 模块新名1][,模块名2 [as 模块新名2]],...
```

**示例**
```python
#导入数学模块
import math
#导入系统模块sys和os模块
```

**作用**

将某模块导入当当前模块

**用法**

模块.属性名

**dir(obj)**

返回模块所有属性的字符串列表

**help(obj)**

查看模块相关的

##from import 语句

**语法**
```python
from 模块名 import 模型属性名1[as 属性新名1][,模块属性名2[as 属性新名2],...]
```

**作用**

将某模块内的一个或多个属性导入到当前模块的作用域

**示例**
```python
from math import pi
from math import pow, sqrt
from math import factorial as fac
```

**from import *语句**

将某模块的所有属性都导入当前模块

```python
from math import *
```

##dir函数

dir([对象]) 返回一个字符串列表

**作用：**

+ 如果没有参数调用，则返回当前作用域内的所有变量的列表
+ 如果给定一个对象作为参数，则返回这个对象的所有变量的列表
    + 对于一个模块，返回这个模块的全部属性
    + 对于一个类对象，返回类对象的所有变量，递归基类对象的所有变量
    + 对于其他对象，返回所有的变量、类变量、基类变量


##时间模块time

**时间简介**

+ 公元纪年从公元0000年1月1日0时开始的
+ 计算机元年是从1970年1月1日0时开始的，此时间为0，之后每过一秒时间
+ UTC时间（Coordinated Universal Time)是从Greenwich时间开始计算的 \
  UTC时间不会因时区问题而产生错误
+ DST阳光节约时间（Daylight Saving Time),又称夏令时，是一个经过

**时间元组**
+ 时间元组是一个9个整数型元素组成的，这9个元素自前至后依次为：
    + 四位的年（如2000）
    + 月（1-12）
    + 日（1-31）
    + 时（0-23）
    + 分（0-59）
    + 秒（0-59）
    + 星期几（0-6，周一是0）
    + 元旦开始日（1-366）
    + 夏令时修正时间（-1，0 or 1).
+ 注：
    + 如果年份值小于100，则会自动转换为加上1900后的值
